# Job Meter with Raine Emulator

## Issue Description

The job meter feature from DATlib does not appear when running Neo Geo games in the Raine emulator, despite attempts to enable developer mode and related settings.

## Attempts Made

- Activated developer mode in Raine.
- Enabled Soft Dip 2-1.
- Tested with a UNIVERSE BIOS.

## Recommendation

Use MAME emulator instead.
